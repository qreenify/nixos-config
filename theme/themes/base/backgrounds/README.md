# Base Theme Wallpaper

Add your preferred wallpaper here. Supported formats:
- PNG
- JPG/JPEG

The theme script will automatically use the first image found in this directory.

For a solid black wallpaper (minimal), you can download from:
- https://www.solidbackgrounds.com/3840x2160-black-solid-color-background.html

Or use any dark/minimal wallpaper you prefer.
