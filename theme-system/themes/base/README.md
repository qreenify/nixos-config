# Base Theme

This is your base theme - a snapshot of your current system colors and setup.

## Color Palette

- **Background**: `#000000` (Pure Black)
- **Foreground**: `#888888` (Gray)
- **Active/Focus**: `#ffffff` (White)
- **Accent**: `#00D4AA` (Cyan/Aqua) - Used for mic active, selection
- **Border Active**: `#33ccff` → `#00ff99` (Cyan to Aqua gradient)
- **Border Inactive**: `#595959aa` (Dark Gray)
- **Urgent/Error**: `#ff6666` (Red)
- **Muted**: `#FF0000` (Red) - Used for mic muted

## Themed Apps

- ✅ **Walker** - Application launcher
- ✅ **Waybar** - Status bar
- ✅ **Hyprland** - Window manager borders
- ✅ **Alacritty** - Terminal colors
- ✅ **Mako** - Notifications

## Usage

To use this theme:
```bash
omarchy-theme-set base
```

Or use the graphical theme browser:
```bash
theme
```

## Notes

This theme was created from your actual running configuration and represents your preferred minimal, dark aesthetic with cyan/aqua accents.
